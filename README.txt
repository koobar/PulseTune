■PulseTuneについて
　PulseTuneは、シンプルな操作で使用できることを目標に開発された、Windows用のオーディオプレイヤーです。
　PulseTuneは無料のソフトウェアです。ユーザーの皆様は、PulseTuneを無償で自由に使用することができます。
　PulseTuneのライセンスについて、詳しくは、LICENSE をお読みください。

　また、PulseTuneで使用しているサードパーティーのライブラリに関するライセンスに関して、詳しくは、
　THIRD-PARTY-LICENSE.txtをお読みください。

■動作環境
　Windows 10 (19H1以降, 64bit), Windows 11
　.NET Framework 4.8 以降が動作する環境

■再生可能なフォーマット一覧
　・AIFF
　・Vorbis
　・FLAC
　・MP2
　・MP3
　・M4A
　・WMA
　・WAV
　・WavPack
　・ZilophiX
　・CDAファイル（オーディオCDトラック）
